# 🍋 Little Lemon – React Native App

A mobile application for **Little Lemon**, a neighborhood bistro, designed to provide a seamless food ordering and menu browsing experience.

---

## 📱 Overview

Little Lemon is a charming restaurant offering simple dishes and classic cocktails in a casual setting. This app brings that experience to mobile by allowing users to:

- Browse menu items  
- Filter dishes by category  
- Search for meals  
- View detailed food descriptions  
- Manage user profile  

The design and branding follow the official Little Lemon style guide.

---

## 🎯 Project Goals

- Deliver a smooth and intuitive mobile experience  
- Implement scalable UI components using React Native  
- Apply modern UX practices (filtering, search, onboarding)  
- Maintain consistency with Little Lemon branding  

---

## 🛠️ Tech Stack

- **React Native**
- **Expo**
- **React Navigation**
- **AsyncStorage**
- **Expo SQLite**

---

## ✨ Features

### 🧭 Navigation
- Stack-based navigation
- Onboarding → Profile → Home/Menu flow

### 🍽️ Menu System
- Dynamic menu display using `FlatList`
- Category-based filtering
- Search functionality

### 👤 User Profile
- Editable user details
- Persistent storage using AsyncStorage

### 🔍 Filtering & Search
- Filter menu by categories
- Real-time search updates

## 🚀 Getting Started

### Prerequisites
- Node.js installed
- Expo CLI installed

### Installation

```bash
git clone https://github.com/jaa38/little-lemon-react-native.git
cd little-lemon-react-native
npm install
```

### Run the App

```bash
npx expo start
```
